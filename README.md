# 🍔 Knight Bite Landing Page Clone

A responsive front-end website inspired by the Knight Bite food delivery platform. This project recreates the design and layout of the original website using HTML and CSS, focusing on modern UI design and responsive web development practices.

## 📖 Project Overview

This project was developed to practice front-end web development skills by recreating the Knight Bite landing page. It demonstrates the use of HTML and CSS to build visually appealing and responsive web interfaces.

## ✨ Features

* Responsive landing page design
* Modern user interface
* Hero banner section
* Franchise promotion section
* App download section
* Social media integration icons
* Clean and organized code structure
* Mobile-friendly layout

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/Piyanshi0411/Demo.git
```

2. Navigate to the project folder

```bash
cd Demo
```

3. Open `Index.html` in your browser.

No additional dependencies or installations are required.
